What is Bugo?

Bugo is a framework for a static website generator called Hugo. Developers can use the base Bugo as a boilerplate to building static websites. Content creators and business can use our prefab Websites with themes and get up and running in no time.

For Content Creators

Use our one click installs to one of our recommended content management solutions. You get a customizable theme and a few suggested pages and sample posts.

Website Features

Bugo comes with the features you come to expect out of a blogging website.

Site Search

Bugo comes with side wide search built in. An index is built by Hugo on every deployment. Because the search is client side it is VERY fast.

SEO Friendly

Bugo has structured data, semantic layouts, breadcrumbs & XML sitemaps.

Taxonomy & Custom Archetypes

Easily create custom archetypes like profiles or products, for example. Then get those pages organized with Tags & Categories

Related Pages

Thought you couldn’t have related items with a static site? Relationships are built automatically by Bugo.

Let's build a website

Let’s build a website!

Great! Please read each step fully before starting. We’re going to teach you some stuff along the way.

Here’s the steps you’ll take to get Bugo up and running.

  1. Setup a repository at GitHub
  2. Select a starter website
  3. Setup Forestry.io CMS
  4. Setup Cloudinary
  5. Setup GitHub Pages
  6. Add your custom domain(optional)
  7. Personalize your website
  8. What’s next?

Setup a GitHub Account

A repository is like a folder for your website. Your websites’s repository contains all of your project’s files and stores each file’s revision history. This means you’ll always have a backup of every version of every file. Neat huh?

Github is where all your content and code is going to live. Use your email address to sign up for an account. We’re also going to use the same account to sign into Forestry.io in future steps.

You’ll need to decide if you want to keep your website’s code and content private or not. GitHub does charge to keep your repository private, but the cost is nominal and you can have as many private repositories as you like. We recommend keeping your repository private.

Once you’re logged into GitHub, stay logged in. You will be using your login to setup other accounts.

Select a Starter Website

We’ve setup a few Bugo starter websites to get you going. Once you’ve decided on a starter, come back here and read the next step before you install your Bugo website.

Setup Forestry.io CMS

This step activates your content management system. So go back your selected starter website’s page and click Install on Forestry.io button. You’ll need to complete the resulting screens. You can use your existing GitHub account to create your forestry.io account, so feel free to. One less password to remember is always a good thing.

This is going to create a repository for you on GitHub, so when you are prompted to give it an appropriate name. If you’re using a custom domain, use that as your repository name. Something like “your-domain-com”. This will take 5-10 minutes to automatically setup.
